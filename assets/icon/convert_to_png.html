사용 방법

  1. 위의 "PNG 다운로드 (1024x1024)" 버튼을 클릭하세요.
  2. 다운로드된 파일들을 다음 위치에 저장하세요:
    • app_icon.png → assets/icon/app_icon.png
    • app_icon_foreground.png → assets/icon/app_icon_foreground.png
  3. 터미널에서 다음 명령어를 실행하세요:
    flutter pub get
    dart run flutter_launcher_icons
  4. 모든 플랫폼의 앱 아이콘이 자동으로 생성됩니다!
