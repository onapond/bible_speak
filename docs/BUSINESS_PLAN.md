# 바이블 스픽 2.0 - 상업화 기획서

## 1. 비전 및 미션

### 비전
"AI 튜터와 함께하는 가장 효과적인 영어 성경 암송 앱"

### 미션
- 기독교인의 성경 암송을 영어 학습과 결합
- AI 기술로 개인화된 발음 교정 및 피드백 제공
- 교회/선교단체 중심의 그룹 학습 커뮤니티 구축

### 타겟 고객
| 세그먼트 | 특징 | 니즈 |
|----------|------|------|
| **핵심** | 20-40대 기독교인, 영어 학습 관심 | 성경 암송 + 영어 실력 향상 |
| **확장** | 교회 주일학교, 청년부 | 그룹 학습, 달란트 경쟁 |
| **특수** | 선교사/유학 준비생 | 영어 성경 유창성 필요 |

## 3. 제품 전략

### 3.1 핵심 가치 제안 (Value Proposition)
```
"하루 10분, AI 튜터와 영어 성경 한 구절"
- 듣고 따라하는 3단계 쉐도잉 학습
- 실시간 발음 교정 (음소 단위 피드백)
- 교회 그룹 랭킹으로 함께 성장
```

### 3.2 핵심 기능 (MVP)

#### A. AI 발음 튜터 (핵심 차별화)
- **Azure Speech** 기반 발음 평가
  - 정확도 (Accuracy): 단어별 정확도
  - 유창성 (Fluency): 끊김 없는 발화
  - 운율 (Prosody): 강세, 억양
- **음소(Phoneme) 레벨 피드백**
  - 한국인이 어려워하는 발음 집중 교정 (th, r, l, f, v)
  - IPA 발음 기호 + 한글 힌트 제공
- **AI 피드백 생성** (Gemini)
  - "이 구절에서 'righteousness'의 강세가 좋았어요!"
  - "th 발음을 혀를 이 사이에 대고 발음해보세요"

#### B. 3단계 쉐도잉 학습
| 단계 | 이름 | 화면 | 통과 조건 |
|------|------|------|----------|
| 1 | Listen & Repeat | 전체 자막 O | 70% 정확도 |
| 2 | Key Expressions | 핵심 단어 빈칸 | 80% 정확도 |
| 3 | Real Speak | 자막 X | 85% 정확도 |

#### C. Speak 스타일 로드맵 UI
```
      ●━━━●━━━◐━━━○━━━○
     1장  2장  3장  4장  5장
    완료 완료 진행중 잠금 잠금
```
- 시각적 진행 상황
- 잠금 해제 동기 부여
- 완료 시 축하 애니메이션

#### D. 그룹/달란트 시스템
- 교회별/소그룹별 그룹 생성
- 주간/월간 달란트 랭킹
- 그룹 목표 달성 시 배지 획득

### 3.3 콘텐츠 로드맵

| Phase | 성경 | 장 수 | 특징 |
|-------|------|-------|------|
| MVP | 말라기, 에베소서, 히브리서 | 23장 | 암송용 인기 구절 |

## 5. 마케팅 전략

### 5.1 Phase 1: 시드 유저 확보 (0-3개월)

#### 전략: 교회 파트너십
- **목표**: 3개 교회, 100명 베타 사용자
- **방법**:
  - 지인 교회 청년부/주일학교 타겟
  - 무료 3개월 프리미엄 제공
  - 피드백 수집 → 제품 개선

#### 전술
1. **카카오톡 오픈채팅방**: "영어성경암송 스터디"
2. **인스타그램**: 영어 성경 구절 카드뉴스
3. **교회 주보/게시판**: 앱 소개 전단

### 5.2 Phase 2: 성장 (3-6개월)

#### 전략: 콘텐츠 마케팅 + 바이럴
- **YouTube**: 영어 성경 발음 교정 영상
  - "한국인이 틀리기 쉬운 성경 영단어 TOP 10"
  - "3분 영어 성경 쉐도잉" 시리즈
- **블로그/브런치**: 영어 암송 학습법 콘텐츠
- **인플루언서**: 기독교 유튜버 협업

#### 바이럴 기능
- 친구 초대 시 프리미엄 7일 무료
- 달란트 랭킹 공유 (인스타 스토리)
- 암송 완료 인증샷 공유

### 5.3 Phase 3: 확장 (6-12개월)

#### 전략: 유료 광고 + 파트너십 확대
- **페이스북/인스타 광고**: 타겟 (기독교 + 영어학습 관심)
- **Google 앱 캠페인**: ASO 최적화
- **선교단체 파트너십**: 선교훈련원, YWAM 등
- **기독교 미디어**: 갓피플, 두란노 배너 광고

### 5.4 ASO (App Store Optimization)

#### 키워드
- 주요: 영어성경, 성경암송, 영어암송, Bible memorization
- 보조: 영어발음, AI튜터, 쉐도잉, 성경공부

#### 앱 제목/부제
- **iOS**: "바이블스픽 - AI 영어성경 암송"
- **Android**: "바이블스픽: 영어성경 암송 | AI 발음 코칭"

├─────────────────────────────────────────────────────────┤
│                    Service Layer                         │

├───────┼──────────┼──────────┼──────────┼───────────────┤
│       ▼          ▼          ▼          ▼               │
│  ┌─────────────────────────────────────────────────┐   │

### 6.2 API 사용량 및 비용 최적화

| API | 무료 티어 | 최적화 전략 |

### 6.3 데이터 모델 (Firestore)

```
Firestore
├── users/{uid}
│   ├── name: string
│   ├── groupId: string
│   ├── talants: number
│   ├── isPremium: boolean
│   ├── subscriptionExpiry: timestamp
│   └── progress/
│       └── {bookId}/
│           └── chapters: {
│                 "1": {
│                   verses: {
│                     "1": {
│                       stage: 1-3,
│                       scores: [85, 78, 92],
│                       bestScore: 92,
│                       completedAt: timestamp
│                     }
│                   }
│                 }
│               }
│
├── groups/{groupId}
│   ├── name: string
│   ├── code: string (초대 코드)
│   ├── createdBy: uid
│   └── memberCount: number
│
└── leaderboard/weekly/{groupId}
    └── rankings: [
          { uid, name, talants, rank }
        ]
```
