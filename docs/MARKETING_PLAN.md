## 2. 타겟 고객 페르소나

### 2.1 핵심 페르소나: 영어 성경 학습자 김은혜 (28세)

| 항목 | 내용 |
|------|------|
| 직업 | 회사원 (대기업 마케팅팀) |
| 신앙 | 교회 청년부 활동 중, 매일 QT |
| 목표 | 영어 실력 향상 + 성경 암송 |
| 고민 | 영어 학원 다닐 시간 없음, 혼자 공부하면 발음 피드백이 없음 |
| 기기 | iPhone 15, AirPods |
| 학습 시간 | 출근길 지하철 (20분), 점심시간 (10분) |
| 지불 의향 | 월 5,000원 이하면 구독 의향 |

**페인 포인트**:
- "영어 성경 암송하고 싶은데 발음이 맞는지 모르겠어요"
- "Speak은 너무 비싸고, 성경 콘텐츠가 없어요"
- "교회에서 함께 암송 챌린지 하고 싶어요"

### 2.2 확장 페르소나: 청년부 리더 박성준 (32세)

| 항목 | 내용 |

## 5. 교회 파트너십 전략

### 5.1 접근 방식

**1단계: 파일럿 교회 확보 (3개월)**
- 목표: 3개 교회, 100명 사용자
- 대상: 지인 네트워크, 중소형 교회 청년부
- 제안: 3개월 무료 그룹 라이선스

**2단계: 사례 구축 (3개월)**
- 성공 사례 문서화
- 교회별 암송 완료율, 참여율 데이터
- 인터뷰 영상 제작

**3단계: 확장 (6개월~)**
- 기독교 연합 행사 참여
- 교단별 접근 (장로회, 감리회 등)
- 선교단체 파트너십 (YWAM, CCC 등)

### 5.2 교회용 제안서 핵심 내용

```
📚 바이블스픽 교회 파트너십 제안

✅ 혜택
• 그룹 라이선스 (20인 기준 월 29,000원)
• 관리자 대시보드 (진척도 리포트)
• 맞춤 성경 범위 설정
• 전용 고객 지원

✅ 도입 사례
• A교회 청년부: 3개월 만에 에베소서 1-2장 암송 완료
• 평균 참여율 78%, 주 3회 학습

✅ 무료 체험
• 1개월 무료 파일럿 프로그램
• 담당자 온보딩 지원
```
